📸 How to Export These as Images

Option A — Browser Screenshot (Easiest)

  1. Open this file in Chrome: map-scrapper/social-assets/index.html
  2. Right-click any graphic → Inspect → find the .card element
  3. Right-click the element → Capture node screenshot
  4. Saves at full resolution (1080×1080, 1080×1350, or 1080×1920)

Option B — DevTools Device Mode

  1. Press F12 → Click the device toolbar icon
  2. Set dimensions to 1080×1080 (or 1080×1920 for stories)
  3. Navigate to each #post1 through #post7
  4. Press Ctrl+Shift+P → type "screenshot" → Capture full size screenshot
